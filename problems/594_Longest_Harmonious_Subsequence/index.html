LeetCode 594. Longest Harmonious Subsequence

We define a harmonious array as an array where the difference between its maximum value and its minimum value is exactly 1.

Given an integer array nums, return the length of its longest harmonious among all its possible subsequences.

 

Example 1:

Input: nums = [1,3,2,2,5,2,3,7]

Output: ; Expected -> 5

Explanation:

The longest harmonious subsequence is [3,2,2,2,3].

Example 2:

Input: nums = [1,2,3,4]

Output: ; Expected -> 2

Explanation:

The longest harmonious subsequences are [1,2], [2,3], and [3,4], all of which have a length of 2.

Example 3:

Input: nums = [1,1,1,1]

Output: ; Expected -> 0

Explanation:

No harmonic subsequence exists.

Example 4:

Input: nums = [-3,-1,-1,-1,-3,-2]

Output: ; Expected -> 4

Example 5:

Input: nums = [10,2,1,4,9,2,3,8,10,8,3,5,8,5,0,8,8,6,0,6,9,8,4,4,5,4,0,1,10,8,10,3,10,2,8,8,7,4,3,7,2,5,3,7,6,5,6,9,1,1,2,7,7,4,10,8,3,-1,7,5,2,7,0,8,4,6,7,3,4,7,1,8,4,1,3,6,0,5,2,10]

Output: ; Expected -> 21
