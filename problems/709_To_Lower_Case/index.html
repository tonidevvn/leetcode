LeetCode 709. To Lower Case

Given a string s, return the string after replacing every uppercase letter with the same lowercase letter.

 

Example 1:

Input: s = "Hello"
Output:  ; Expected
          -> "hello"

Example 2:

Input: s = "here"
Output:  ; Expected
          -> "here"

Example 3:

Input: s = "LOVELY"
Output:  ; Expected
          -> "lovely"

 

Constraints:

  • 1 <= s.length <= 100
  • s consists of printable ASCII characters.
